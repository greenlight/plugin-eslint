# greenlight/eslint [![License][license-image]][license-url] [![Build Status][travis-image]][travis-url]

> [ESLint][] plugin for [Greenlight][]

## Usage

in your `greenlight.yml` file include:

```js
plugins:
  greenlight/eslint: true
```
